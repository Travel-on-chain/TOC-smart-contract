# Travel on chain contracts

## Clone this repo

```shell
git clone https://github.com/Travel-on-chain/contract-core.git
```

## Install dependencies

```shell
make remove
make install
```

## Build contracts

```shell
make build
```

## Test contracts

```shell
make test
```

# Start[local-env]:

# Step1 for local 
make deploy-anvil
# Strp2 for local
make upgrade
